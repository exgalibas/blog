## 必装插件

### 1. GitHub Copilot
AI 代码补全，写注释就能生成代码，效率提升明显。

### 2. Error Lens
把错误和警告直接显示在代码行旁边，不用看底部面板。

### 3. GitLens
查看每一行的 Git 历史，谁改的、什么时候改的，一目了然。

## 代码质量

### 4. ESLint
代码规范检查，配合项目配置自动修复。

### 5. Prettier
代码格式化，团队统一风格的神器。

## 主题与外观

### 6. One Dark Pro
经典暗色主题，长时间使用不累眼。

### 7. Icon Theme (Material Icon)
文件图标美化，快速识别文件类型。

## 我的设置

```json
{
  "editor.fontSize": 14,
  "editor.tabSize": 2,
  "editor.formatOnSave": true,
  "editor.minimap.enabled": false,
  "editor.wordWrap": "on"
}
```

## 小技巧

- `Cmd + P`：快速打开文件
- `Cmd + Shift + F`：全局搜索
- `Cmd + D`：选中下一个相同词
- `Alt + 上下`：移动行
