## 启动器

### Raycast
替代 Spotlight，插件生态丰富。我常用：
- 剪贴板历史
- 窗口管理
- 快速翻译
- AI 对话

## 窗口管理

### Rectangle
免费开源的窗口管理工具，拖拽到边缘自动分屏。

快捷键：
- `Ctrl + Option + ←` 左半屏
- `Ctrl + Option + →` 右半屏
- `Ctrl + Option + ↑` 全屏

## 终端

### iTerm2 + fish shell
- iTerm2：分屏、搜索、自动补全
- fish：开箱即用的智能补全和语法高亮

## 编辑器

### Neovim
终端里的编辑器，适合快速编辑配置文件。

## 其他

| 工具 | 用途 |
|------|------|
| IINA | 视频播放器 |
| Keka | 解压缩 |
| DevToysMac | 开发者小工具集 |
| TablePlus | 数据库客户端 |

## 总结

工具只是手段，关键是找到适合自己的工作流。不建议一次装太多，按需添加。
