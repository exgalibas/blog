# 使用 Next.js 搭建博客

最近用 Next.js 搭了一个博客，整体体验不错。

## 技术选型

- **框架**：Next.js 14 (App Router)
- **样式**：Tailwind CSS
- **内容**：Markdown + Frontmatter
- **部署**：Vercel

## 为什么选 Next.js

1. SSG 支持好，博客天然适合静态生成
2. 文件系统路由简洁直观
3. 部署到 Vercel 零配置

## 总结

Next.js 做博客确实方便，推荐试试。
