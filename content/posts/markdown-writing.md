# 使用 Markdown 写作

Markdown 是一种轻量级标记语言，由 John Gruber 于 2004 年创造。它的设计理念是让文档易读易写，同时可以转换为 HTML。

## 基本语法

### 标题

使用 `#` 号表示标题层级：

```
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **加粗**：`**文字**`
- *斜体*：`*文字*`
- `行内代码`：反引号包裹

### 列表

无序列表使用 `-` 或 `*`：

- 第一项
- 第二项
- 第三项

有序列表使用数字：

1. 第一步
2. 第二步
3. 第三步

### 代码块

```javascript
function greeting(name) {
  return `Hello, ${name}!`;
}

console.log(greeting('World'));
```

### 引用

> 写作是思考的延伸。当你开始写，你就在思考。
> — 某位智者

### 表格

| 功能 | 语法 | 示例 |
|------|------|------|
| 加粗 | `**text**` | **bold** |
| 斜体 | `*text*` | *italic* |
| 代码 | `` `code` `` | `code` |

## 为什么选择 Markdown

1. **专注内容**：不需要考虑排版
2. **纯文本**：兼容任何编辑器
3. **版本控制友好**：Git diff 清晰
4. **广泛支持**：GitHub、Notion、各种博客平台

## 推荐工具

- **编辑器**：VS Code + Markdown Preview
- **命令行**：Typora、Obsidian
- **在线**：StackEdit、Dillinger

开始用 Markdown 写作吧！
